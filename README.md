# OrangeHRM-Manual
Manual testing project for OrangeHRM: test plan, test scenarios, test cases, bug reports, and execution results.
WHAT IS ORANGE HRM?
IT IS A B2B APPLICATION AND IT IS A MEDIUM LEVEL APPLICATION.
NOTE:    B2B STANDS FOR ‘BUSINESS TO BUSINESS’.
         Orange HRM is a B2B Human Resource Management (HRM/HRIS) web application used by organizations to manage employee and HR operations in a centralized system. It               supports secure access, structured workflows, and HR reporting.

WHAT IS PROJECT OVERVIEW?
THIS PROJECT AIMS TO UNDERSTAND THE ORANGEHRM APPLICATION FLOW AND VALIDATE THE MAIN HR OPERATIONS USING MANUAL TESTING (AND BASIC AUTOMATION IF REQUIRED). THE FOCUS IS ON VERIFYING CORE FUNCTIONALITIES SUCH AS LOGIN, EMPLOYEE MANAGEMENT (PIM), AND LEAVE MANAGEMENT, ALONG WITH BASIC ADMIN CONFIGURATIONS.
